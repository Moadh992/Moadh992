## What I'm working on

### [churn-agent](https://github.com/Moadh992/churn-agent) — Voice API Churn Detection System

Production multi-agent pipeline that monitors voice API accounts across three correlated signals and autonomously triggers recovery actions before customers cancel.

```
Support tickets ──► Ticket Agent (LLM classifier, Langfuse traced)
API error logs  ──► Error Agent  (pure Python, 3× baseline rule)      ──► Supervisor ──► Tier Router
Usage time-series ► Usage Agent  (ClickHouse sliding window SQL)            │                   │
                                                                     2+ signals         SMB: auto-credit
                                                                     in 72h → HIGH      Enterprise: Slack
```

Built in under 2 weeks, solo, from scratch.
Stack: `LangGraph` `Temporal.io` `ClickHouse` `Langfuse` `Grafana` `Ollama` `Slack SDK` `Pydantic v2`
